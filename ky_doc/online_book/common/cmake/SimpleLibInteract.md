# CXX子模块间的调用

::: danger

cxx子模块间的相互调用请小心，如果出现`A need B` 同时`B need A`时，会出现递归include，编译报错

最好的处理方式，是将两者共同的东西进行抽离，否则使用一个预编译指令，让这个文件只能在编译的时候加入一次语法树

:::
